# **App Name**: ChatAssist

## Core Features:

- AI Chat Interface: Provides an intuitive conversational interface for users to interact with the AI assistant, featuring input and output display for natural communication.
- Gemini Flash 2.0 Integration: Utilizes the Gemini Flash 2.0 API to power intelligent responses and provide assistance based on user queries, optimized for speed.
- Dynamic Chat Summarizer Tool: An AI-powered tool that summarizes the ongoing chat history, selectively deciding which key information to retain and synthesize. This optimizes context handling and reduces token usage for long conversations, effectively managing memory.
- Text-to-Speech Output: Converts AI generated text responses into natural-sounding speech for an auditory user experience, using a voice profile customized for Spinneys' brand.
- Chat History Persistence: Stores user chat history, including both user inputs and AI responses, allowing for continuous conversation context and later review.
- Message Stream Display: Displays the conversation in a familiar chat-bubble format, clearly differentiating user and AI messages.

## Style Guidelines:

- Primary brand color: A warm, bright yellow (#FFD700) reflecting energy and freshness, used for key interactive elements and branding accents.
- Background color: A clean and crisp white (#FFFFFF) to provide a spacious and modern canvas, contrasting beautifully with the yellow.
- Accent color: An energetic orange-yellow (#FFA500) for calls to action, active states, and highlights, symbolizing vibrancy and promptness.
- All text will use 'Inter', a grotesque-style sans-serif font. Its modern, machined, and neutral aesthetic ensures high readability and a clean professional feel suitable for both headlines and body text.
- Use modern, outlined icons that convey clarity and functionality. Focus on icons relevant to chat (send, microphone), AI (sparkle, brain), and user actions, maintaining a consistent style.
- A clean, two-column layout with the chat history taking center stage, flanked by optional contextual information or settings. Responsive design to ensure usability across devices.
- Subtle, fluid animations for message sending and receiving, loading states, and when the AI's TTS feature is active, enhancing responsiveness and user feedback.
